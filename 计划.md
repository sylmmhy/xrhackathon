│ New file: yume-server/modules/meshy.py                                                                                   │
│                                                                                                                          │
│ 仿照 modules/marble.py 的 MarbleClient 模式：                                                                            │
│ - MeshyClient 类，Authorization: Bearer {key} 认证头                                                                     │
│ - create_image_to_3d(image_data) → POST /openapi/v1/image-to-3d，返回 task_id                                            │
│   - image_data 为 base64 data URI（data:image/png;base64,...），直接传给 image_url 字段，无需公网 URL                    │
│ - poll_task(task_id, timeout=300) → GET 轮询，间隔 5s，检查 status                                                       │
│ - download_glb(glb_url, output_path) → 下载 GLB 到本地                                                                   │
│ - image_to_3d(image_data, output_path, on_progress) → 编排全流程                                                         │
│                                                                                                                          │
│ 请求参数:                                                                                                                │
│ {                                                                                                                        │
│   "image_url": "data:image/png;base64,...",                                                                              │
│   "ai_model": "meshy-6",                                                                                                 │
│   "should_texture": true,                                                                                                │
│   "enable_pbr": true,                                                                                                    │
│   "target_polycount": 30000,                                                                                             │
│   "remove_lighting": true                                                                                                │
│ }                                                                                                                        │

│                                                                                                                          │

│ POST /api/world/{world_id}/objects — 上传图片，启动 Meshy 生成                                                           │
│ - 验证 world 存在且 status == "complete"                                                                                 │
│ - 接受 multipart (file field "image") 或 JSON base64                                                                     │
│ - 保存原图到 /assets/{world_id}/object_{id}_source.png                                                                   │

│                                                                                                                          │

│ - 返回 object dict (含 status, progress, glb_url, error)                                                                 │
│                                                                                                                          │
│ GET /api/world/{world_id}/objects — 列出世界中所有物体（含已完成的 glb_url）                                             │
│ - 返回 {world_id, objects: [...]}                                                                                        │
│ - 关键用途: 页面刷新后，前端调用此接口重新加载所有已生成的物体                                                           │

│ 关键注意事项                                                                                                             │
│                                                                                                                          │
│ - 无需公网 URL: 图片以 base64 data URI 传给 Meshy，无需 ngrok 等隧道                                                     │
│ - CORS: Yume 已配置 allow_origins=["*"]，前后端跨域无问题                                                                │
│ - 生成时间: Meshy image-to-3d 约 1-3 分钟，poll timeout 300s                                                             │
│ - 模型缩放: Meshy 输出大小不一，前端自动归一化到 ~0.5m                                                                   │
│ - 刷新持久化: 页面刷新后通过 GET /objects + loadExistingObjects() 重新加载所有物体                                       │
│ - 失败重试: UI 提供重试按钮，用同一张图重新提交                                                                          │
